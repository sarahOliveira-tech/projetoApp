Principais elementos de junção entre HTML e CSS

Para estilizar uma página HTML, indica-se o uso de um arquivo CSS externo. Há também a possibilidade de estilizá-lo em linha, porém não é uma boa prática no HTML 5

EM LINHA

<p style="color: ##027fe9;"> Exemplo de código </p>

EM ARQUIVO CSS


    p{
        color: #027fe9;    
    }

Quando lincamos um aqrvivo CSS na página HTML, todos os elementos que forem estilizados no arquivo CSS serão estilizados na página HTML

No código de exmplo da estiização em arquivo externo, todos os elementos <p> no HTML serão estilizados da maneira que foram designados no arquivo CSS

Uma forma de estilizar elementos iguais de maneiras diferentes é usando containers

Tabela sobre elementos containers e de estilização
Id Usado para identificar um elemento
Class Agrupa vários elemntos para estilização e comportamento
Div Agrupa e organiza o conteúdo para fins de estilização e estruturação
