Principais Tags HTML

Abaixo estão exemplos das principais tags usadas no site:

1. Título (h1, h2, etc.)

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

2. Parágrafo (p)

<p>Este é um parágrafo de exemplo.</p>

3. Lista não ordenada (ul) e itens de lista (li)


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

4. Imagem (img)

<img src="caminho/para/imagem.png" alt="Descrição da imagem">

5. Tabela (table)


<table>
    <thead>
        <tr>
            <th>Cabeçalho 1</th>
            <th>Cabeçalho 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Dado 1</td>
            <td>Dado 2</td>
        </tr>
    </tbody>
</table>

6. Link (a)

<a href="https://exemplo.com">Texto do Link</a>
